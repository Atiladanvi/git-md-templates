# Nome do Projeto

## Requerimentos 

* PHP >= 7.2.5
* Composer PHP >= 1.9.0

Extensões do php:

* OpenSSL
* PDO
* Mbstring
* Tokenizer
* XML
* Ctype
* JSON

Bancos suportados

 * MySQL 5.6+
 * PostgreSQL 9.4+
 * SQLite 3.8.8+
 * SQL Server 2017+

## Instalação

Clone o repositório
  
    git clone https://github.com/autor/projeto.git
    
Adentre à pasta do projeto

    cd /projeto

Instale as dependências do projeto

    composer install            

Crie um arquivo de configuração de ambiente

    cp env.example .env
    
Gere uma chave randômica para a aplicação

    php artisan key:generate    
    
Configure as variáveis no arquivo `.env`

    APP_ENV=<local|production>
    
    DB_CONNECTION=<mysql|pgsql|sqlite|sqlsrv>
    DB_HOST=<host>
    DB_PORT=<port>
    DB_DATABASE=<databse name>
    DB_USERNAME=<database user>
    DB_PASSWORD=<database password>
    
Migre e popule o banco de dados

    php artisan migrate --seed    
    
Execute a aplicação
    
    php artisan serve
    
Veja    
 
http://localhost:8080
